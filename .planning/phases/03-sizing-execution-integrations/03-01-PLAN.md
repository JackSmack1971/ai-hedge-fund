| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | risk_manager.py confidence_multiplier read | mitigate | Clamp with max(0.0, min(1.0, float(val) if math.isfinite(float(val)) else 1.0)); prevents NaN/Inf/negative propagation into position_limit |
| T-03-02 | Tampering | risk_manager.py size_multiplier read | mitigate | Same clamp as T-03-01; negative size_multiplier from corrupt state cannot produce negative position_limit |

| T-03-SC | Tampering | npm/pip/cargo installs | accept | No new packages installed in Phase 3; all dependencies already present in poetry.lock |
